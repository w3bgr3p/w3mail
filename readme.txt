Бот для мониторинга сообщений и уведомлений на основе ключевых слов.

Команды:

/start - Запуск бота и вывод списка доступных команд.

/add_k [alrt/addr] [ключ] - Добавление нового ключа в выбранный словарь.
Пример: /add_k alrt фрукты

/add_v [alrt/addr] [ключ] [значение1] [значение2] ... - Добавление значений к существующему ключу в выбранном словаре.
Пример: /add_v alrt фрукты яблоко груша

/rm_k [alrt/addr] [ключ] - Удаление ключа и всех его значений из выбранного словаря.
Пример: /rm_k alrt фрукты

/rm_v [alrt/addr] [ключ] [значение1] [значение2] ... - Удаление выбранных значений из ключа в выбранном словаре.
Пример: /rm_v alrt фрукты яблоко

/edit_k [alrt/addr] [старый_ключ] [новый_ключ] - Изменение имени ключа в выбранном словаре.
Пример: /edit_k alrt фрукты овощи

/edit_v [alrt/addr] [ключ] [старое_значение] [новое_значение] - Изменение значения у выбранного ключа в словаре.
Пример: /edit_v alrt фрукты яблоко апельсин

/view [alrt/addr] - Вывод содержимого выбранного словаря или обоих словарей.
Примеры: 
/view alrt - выводит только словарь alerts
/view addr - выводит только словарь addresses
/view - выводит оба словаря

/help - Вывод этого руководства.

При обнаружении ключевого слова из словаря alerts в сообщении, бот будет искать соответствующие адреса из словаря addresses и отправлять уведомление.

Примечание: 
- alrt относится к словарю alerts, который используется для мониторинга ключевых слов.
- addr относится к словарю addresses, который содержит адреса для уведомлений.
